Overview

The CNC Plotter / Laser Cutter is a multi-axis computer-controlled machine designed for precise plotting, engraving, and laser cutting operations.
This project focuses on innovation, affordability, and flexibility, making advanced CNC technology accessible for education, prototyping, and small-scale manufacturing.

Problem Statement

Commercial multi-axis CNC machines and laser cutters are:
Very expensive
Complex to maintain
Not easily accessible for students, makers, and small workshops
This project aims to bridge the gap by developing a low-cost, customizable CNC Plotter / Laser Cutter with multi-axis capability.
